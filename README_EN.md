## Quick Start

```bash
curl -sSL https://raw.githubusercontent.com/Aaowu/weclaw/main/install.sh | sh
weclaw start
```

## What this fork adds

- global Hermes detection and ACP integration
- Hermes-native command passthrough when Hermes is the active agent
- better documentation for Hermes custom model setups such as `CLI Proxy API`

Hermes itself stays upstream. Install and update Hermes normally, then let WeClaw connect to `hermes acp`.
